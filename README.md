## Specifications
 - Ruby 2.7.2
 - Rails 6.1.2.1 or newer

## Execution
In order to run the project, simply build the image with docker-compose
```
docker-compose build
```
run it
```
docker-compose up
```
and visit it on `http://localhost:3000/`.
